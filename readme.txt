درگاه پرداخت پی استار برای افزونه گرویتی فرمز وردپرس
برای درخواست پذیرندگی و دریافت اطلاعات ترمینال از لینک زیر اقدام نمایید :
https://my.paystar.ir/

== Description ==

with this plugin, you can easily add paystar.ir gateway to gravityforms checkout system
با این پلاگین میتونید درگاه پرداخت پی استار رو برای دریافت مبالغ تعریف شده به فرم ساز گرویتی فرمز اضافه کنید


== Installation ==
 
just enter paystar.ir terminal (API-Key)
راه اندازی و پیکربندی افزونه نیاز به تنظیمات خاصی نیست. فقط کافیه کد ترمینال پذیرندگی خودتون رو در تنظیمات افزونه وارد کنید
برای ورود به صفحه تنظیمات افزونه از منوی گرویتی فرمز ، زیرمنوی پی استار کلیک کنید ؛ سپس روی دکمه آبی رنگ تنظیمات درگاه پی استار کلیک کنید
